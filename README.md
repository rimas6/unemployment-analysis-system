#  Parallel Unemployment Analysis System

A C-based project that analyzes unemployment data using parallel computing techniques.

It compares performance between:
- Serial execution
- OpenMP (shared memory)
- MPI (distributed memory)

---

##  Tools
- C
- OpenMP
- MPI (Microsoft MPI)
- GCC

---

##  Dataset
Unemployment dataset (urban & rural areas)

Source: https://rshiny.ilo.org/dataexplorer58/?lang=en&segment=indicator&id=UNE_TUNE_SEX_AGE_GEO_NB_Q

---

##  Goal
Compare performance of serial vs parallel processing for large-scale data analysis.
